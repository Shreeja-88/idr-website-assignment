# Institute of Digital Risk (IDR) – Website Assignment

This project is a responsive homepage website designed for the **Institute of Digital Risk (IDR)**.
The goal of this assignment was to design a modern landing page that represents the institute’s focus on **digital risk, cybersecurity, and technology governance**.

## Features

* Responsive modern website layout
* Sticky navigation bar
* Startup-style hero section
* Service pillars with hover animation
* Community information section
* Contact form
* Smooth scrolling navigation
* Clean and minimal UI design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts

## Project Structure

```
IDR-Website
│
├── index.html
├── style.css
├── script.js
└── logo.png
```

## Website Sections

1. **Hero Section** – Introduction to the Institute of Digital Risk
2. **About Section** – Overview of the institute and its purpose
3. **Service Pillars**

   * Academy
   * Innovation
   * Advisory
4. **Community Section** – Describes the IDR community
5. **Contact Section** – Simple form for inquiries

## Design Concept

The design uses a **minimal color palette of orange, black, and white** to represent innovation, resilience, and professionalism.
The cube-based logo symbolizes **structure, security, and layered digital risk management**.

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a web browser
